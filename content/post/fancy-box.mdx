This Combobox is inspired by [GitHub](https://github.com)'s PR label selector and is powered by [shadcn/ui](https://ui.shadcn.com).

It uses the following components:

- [Accordion](https://ui.shadcn.com/docs/components/accordion)
- [Badge](https://ui.shadcn.com/docs/components/badge)
- [Button](https://ui.shadcn.com/docs/components/button)
- [Command](https://ui.shadcn.com/docs/components/command)
- [Dialog](https://ui.shadcn.com/docs/components/dialog)
- [Input](https://ui.shadcn.com/docs/components/input)
- [Label](https://ui.shadcn.com/docs/components/label)
- [Popover](https://ui.shadcn.com/docs/components/popover)

The **only** change to the core files was adding a smaller `Button` size:

```js
size: {
  default: "h-10 py-2 px-4",
  xs: "h-8 px-2 rounded-md", // added
  // ...
},
```

The source code is available on [GitHub](https://github.com/mxkaske/mxkaske.com/tree/main/components/craft/fancy-box.tsx).
